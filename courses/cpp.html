Overview

C++ was created by Bjarne Stroustrup at Bell Labs in 1985 as an extension of C with classes. It evolved through C++98, C++11 (major modern update), C++14, C++17, C++20, and C++23, each adding powerful features while maintaining backward compatibility and zero-cost abstractions.

C++ is a multi-paradigm language supporting procedural, object-oriented, generic, and functional programming. It gives developers fine-grained control over memory and system resources while providing high-level abstractions like templates, STL containers, smart pointers, lambdas, and ranges.

The language dominates in game development (Unreal Engine), high-frequency trading, web browsers (Chrome, Firefox), databases (MongoDB), operating systems, embedded systems, machine learning frameworks (TensorFlow), and 3D graphics (OpenGL, Vulkan).

Notes

1. RAII (Resource Acquisition Is Initialization)

RAII ties resource lifetime to object lifetime. Resources are acquired in constructors and released in destructors. Smart pointers implement RAII for memory.

void process() {
    std::ifstream file("data.txt"); // RAII: opens file
    std::vector data(100);      // RAII: allocates heap
    std::unique_ptr ptr(new int[50]); // RAII: owner
    // ... work ...
} // file closes, vector freed, ptr deleted automatically!

2. Smart Pointers

// unique_ptr: sole ownership
auto ptr = std::make_unique(42);
// auto p2 = ptr; // Error: no copy

// shared_ptr: shared ownership
auto s1 = std::make_shared(10);
auto s2 = s1; // Both share ownership
// Memory freed when last shared_ptr dies

// weak_ptr: non-owning observer
std::weak_ptr w = s1;
if (auto shared = w.lock()) {
    // Use shared if object still alive
}

3. Move Semantics

class Buffer {
    char* data; size_t size;
public:
    Buffer(size_t n) : size(n), data(new char[n]) {}
    ~Buffer() { delete[] data; }

    // Move constructor - steals resources
    Buffer(Buffer&& other) noexcept
        : data(other.data), size(other.size) {
        other.data = nullptr;
        other.size = 0;
    }

    // Move assignment
    Buffer& operator=(Buffer&& other) noexcept { ... }
};

Buffer createBuffer() { return Buffer(1024); }
Buffer b = createBuffer(); // Move, not copy!

4. STL Algorithms

std::vector v = {5, 2, 8, 1, 9, 3, 7};

std::sort(v.begin(), v.end());  // 1,2,3,5,7,8,9
auto it = std::find(v.begin(), v.end(), 5);
auto count = std::count_if(v.begin(), v.end(),
    [](int n) { return n > 5; });

// Chaining with ranges (C++20)
auto result = v | std::views::filter([](int n) { return n % 2 == 0; })
                | std::views::transform([](int n) { return n * n; });

5. Lambda Expressions

auto add = [](int a, int b) { return a + b; };
std::cout << add(3, 4); // 7

int multiplier = 3;
auto times = [multiplier](int x) { return x * multiplier; };
std::cout << times(5); // 15

// Mutable lambda
int counter = 0;
auto increment = [counter]() mutable { return ++counter; };
// Each call creates a new copy of counter

6. constexpr & Compile-Time

constexpr int factorial(int n) {
    return n <= 1 ? 1 : n * factorial(n - 1);
}
int arr[factorial(5)]; // Compile-time! arr[120]

constexpr double PI = 3.14159;

// consteval (C++20): always evaluated at compile time
consteval int square(int n) { return n * n; }
int x = square(5); // Always compile-time

Examples

Example: Thread-safe Queue

template
class ThreadSafeQueue {
    std::queue queue;
    mutable std::mutex mtx;
    std::condition_variable cv;
public:
    void push(T value) {
        std::lock_guard lock(mtx);
        queue.push(std::move(value));
        cv.notify_one();
    }

    T pop() {
        std::unique_lock lock(mtx);
        cv.wait(lock, [this] { return !queue.empty(); });
        T value = std::move(queue.front());
        queue.pop();
        return value;
    }
};
