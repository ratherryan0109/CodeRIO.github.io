JavaScript

Overview

JavaScript was created by Brendan Eich in 1995 at Netscape and has grown into one of the most popular and versatile programming languages in the world. It enables interactive web pages, server-side applications (Node.js), mobile apps (React Native), desktop apps (Electron), and even IoT device programming.

JavaScript is a high-level, interpreted, multi-paradigm language supporting object-oriented, functional, and event-driven programming styles. Its ecosystem includes powerful frameworks like React, Angular, and Vue.js for frontend, Express.js and Next.js for backend, and TypeScript for type safety.

Modern JavaScript (ES6+) introduced classes, arrow functions, template literals, destructuring, promises, async/await, modules, and more. The language continues to evolve yearly with new features that improve developer experience and code quality.

Notes

1. Closures

A closure is a function that retains access to its outer scope even after the outer function has returned. Used for data privacy, function factories, and maintaining state.

function counter() {
  let count = 0;
  return function() {
    count++;
    return count;
  };
}
const myCounter = counter();
console.log(myCounter()); // 1
console.log(myCounter()); // 2

2. Promises & Async/Await

Promises represent values that may be available now, later, or never. async/await provides syntactic sugar over promises for cleaner asynchronous code.

// Promise
fetch('https://api.example.com/data')
  .then(res => res.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

// Async/Await
async function getData() {
  try {
    const res = await fetch('https://api.example.com/data');
    const data = await res.json();
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}

3. The this Keyword

this refers to the execution context. In methods it refers to the object, in regular functions it's window/undefined (strict mode), arrow functions inherit this from parent scope.

const obj = {
  name: 'Alice',
  greet() { console.log(`Hi, I'm ${this.name}`); },
  arrowGreet: () => console.log(`Hi, I'm ${this.name}`)
};
obj.greet();       // Hi, I'm Alice
obj.arrowGreet();  // Hi, I'm undefined (arrow inherits from window)

4. Prototypal Inheritance

Every JS object has a prototype. Properties/methods are looked up through the prototype chain. ES6 classes are syntactic sugar over prototypes.

function Animal(name) { this.name = name; }
Animal.prototype.speak = function() {
  return `${this.name} makes a sound.`;
};

class Dog extends Animal {
  speak() { return `${this.name} barks!`; }
}

const dog = new Dog('Rex');
console.log(dog.speak()); // Rex barks!

5. Event Delegation

Instead of adding event listeners to many child elements, add one listener to a parent and use event.target to identify which child triggered it. More efficient for dynamic lists.

document.querySelector('ul').addEventListener('click', (e) => {
  if (e.target.tagName === 'LI') {
    console.log('Clicked:', e.target.textContent);
  }
});

6. Array Methods (map, filter, reduce)

These functional array methods replace traditional for loops for data transformation, filtering, and aggregation.

const nums = [1, 2, 3, 4, 5];
const doubled = nums.map(n => n * 2);      // [2,4,6,8,10]
const evens = nums.filter(n => n % 2 === 0); // [2,4]
const sum = nums.reduce((acc, n) => acc + n, 0); // 15

7. Modules (import/export)

ES6 modules allow splitting code across files. Named exports for multiple exports, default exports for single main export.

// math.js
export const PI = 3.14159;
export function add(a, b) { return a + b; }
export default class Calculator { ... }

// app.js
import Calculator, { PI, add } from './math.js';

8. Truthy & Falsy Values

Falsy values: false, 0, '' (empty string), null, undefined, NaN. Everything else is truthy. Important for conditionals and logical operators.

const name = '';
if (name) { console.log('Has name'); }
else { console.log('No name'); } // No name

// Short-circuit evaluation
const config = { theme: 'dark' };
const theme = config.theme || 'light'; // 'dark'

Examples

Example 1: To-Do List App

const todoInput = document.getElementById('todo-input');
const todoList = document.getElementById('todo-list');

function addTodo() {
  const text = todoInput.value.trim();
  if (!text) return;

  const li = document.createElement('li');
  li.textContent = text;
  li.addEventListener('click', () => li.classList.toggle('done'));
  li.addEventListener('dblclick', () => li.remove());

  todoList.appendChild(li);
  todoInput.value = '';
}

document.getElementById('add-btn').addEventListener('click', addTodo);
todoInput.addEventListener('keypress', (e) => {
  if (e.key === 'Enter') addTodo();
});

Example 2: Fetch API & Data Display

async function displayUsers() {
  const container = document.getElementById('users');
  container.innerHTML = '
Loading...
';

  try {
    const res = await fetch('https://jsonplaceholder.typicode.com/users');
    const users = await res.json();

    container.innerHTML = users.map(user => `
      

        
${user.name}

        
${user.email}

        
${user.company.name}

      

    `).join('');
  } catch (err) {
    container.innerHTML = `
Error: ${err.message}
`;
  }
}

displayUsers();
