Pandas

Pandas is a powerful, open-source Python library for data manipulation and analysis. Built on top of NumPy, it provides easy-to-use data structures like Series and DataFrame for working with structured data, making it an essential tool for data scientists and analysts worldwide.

Overview

Pandas was created by Wes McKinney in 2008 to provide flexible, high-performance data analysis tools in Python. The name Pandas derives from "Panel Data", an econometrics term. Its two primary data structures - Series (1-dimensional labeled array) and DataFrame (2-dimensional labeled data structure with columns of potentially different types) - make it intuitive for users familiar with spreadsheets or SQL.

Pandas excels at data ingestion (CSV, Excel, SQL, JSON, HTML, Parquet), data cleaning (handling missing values, duplicates, outliers), data transformation (grouping, merging, reshaping, pivoting), time series analysis (date ranges, resampling, rolling windows), and integration with visualization libraries like Matplotlib and Seaborn. It is the foundation of the Python data science stack.

Pandas is used by Netflix (content recommendations), Spotify (music analytics), JP Morgan (financial modeling), Uber (ride data analysis), Airbnb (pricing algorithms), NASA (scientific data), and countless data science teams worldwide. It is considered a must-have skill for any data professional working in Python.

Notes

1. DataFrame Basics

DataFrame is a 2D labeled data structure with columns of potentially different types. It is the primary Pandas data structure.

import pandas as pd

# From dict
data = {'name': ['Alice', 'Bob', 'Charlie'],
        'age': [25, 30, 35],
        'city': ['NYC', 'LA', 'Chicago']}
df = pd.DataFrame(data)

# From CSV
df = pd.read_csv('data.csv')

# Basic info
df.head()    # First 5 rows
df.info()    # Column types and null counts
df.describe() # Summary statistics

2. Indexing with loc and iloc

loc uses label-based indexing, iloc uses integer position-based indexing.

# loc - label based
df.loc[0]              # First row by index label
df.loc[:, 'name']      # All rows, 'name' column
df.loc[0:2, ['name', 'age']]  # Rows 0-2, specific columns

# iloc - integer position
df.iloc[0]             # First row
df.iloc[:, 0]          # First column
df.iloc[0:3, 0:2]      # First 3 rows, first 2 columns

# Boolean indexing
df[df['age'] > 30]     # Rows where age > 30
df.query('age > 30')   # Same using query string

3. Handling Missing Data

Pandas represents missing data as NaN (Not a Number) and provides tools to detect, remove, or fill missing values.

df.isnull().sum()      # Count nulls per column
df.isna().sum()        # Same as isnull

# Remove missing values
df.dropna()            # Drop rows with any NaN
df.dropna(axis=1)      # Drop columns with any NaN
df.dropna(thresh=3)    # Drop rows with less than 3 non-null

# Fill missing values
df.fillna(0)           # Fill with 0
df.fillna(df.mean())   # Fill with column mean
df.fillna(method='ffill')  # Forward fill
df.fillna(method='bfill')  # Backward fill

4. GroupBy Operations

GroupBy splits data into groups, applies a function, and combines results - a powerful pattern for aggregation.

# Basic grouping
df.groupby('city').mean()
df.groupby('city')['age'].agg(['mean', 'std', 'count'])

# Multiple groupby
df.groupby(['city', 'gender']).agg({
    'age': 'mean',
    'salary': 'sum'
})

# Transform (same shape as original)
df['age_zscore'] = df.groupby('city')['age'].transform(
    lambda x: (x - x.mean()) / x.std()
)

5. Merging DataFrames

Similar to SQL joins, Pandas provides merge() for combining DataFrames on keys.

# SQL-style joins
pd.merge(df1, df2, on='key')
pd.merge(df1, df2, on='key', how='inner')
pd.merge(df1, df2, on='key', how='left')
pd.merge(df1, df2, on='key', how='outer')

# Merge on different column names
pd.merge(df1, df2, left_on='id', right_on='user_id')

# Concatenation
pd.concat([df1, df2])              # Row-wise
pd.concat([df1, df2], axis=1)      # Column-wise

6. Time Series Analysis

Pandas has extensive support for time series data with date ranges, resampling, and rolling windows.

# Create date range
dates = pd.date_range('2023-01-01', periods=10, freq='D')

# Set datetime index
df.index = pd.to_datetime(df['date'])

# Resampling
df.resample('M').mean()      # Monthly average
df.resample('W').sum()       # Weekly sum

# Rolling window
df['rolling_avg'] = df['value'].rolling(window=7).mean()
df['expanding'] = df['value'].expanding().mean()

# Shift and diff
df['prev_day'] = df['value'].shift(1)
df['daily_change'] = df['value'].diff()

import pandas as pd

# Sample sales data
sales = pd.DataFrame({
    'date': pd.date_range('2024-01-01', periods=100, freq='D'),
    'product': ['A', 'B', 'C'] * 33 + ['A'],
    'revenue': np.random.randint(100, 1000, 100),

})

# Monthly revenue by product
monthly = sales.groupby([pd.Grouper(key='date', freq='M'), 'product']).agg({
    'revenue': 'sum',
    'units': 'sum'
}).reset_index()

# Top products by revenue
top_products = sales.groupby('product')['revenue'].sum().sort_values(ascending=False)

# Moving average
sales['revenue_ma7'] = sales.groupby('product')['revenue'].transform(
    lambda x: x.rolling(7).mean()
)

Example 2: Data Cleaning Pipeline

A complete data cleaning workflow for a messy dataset.

def clean_dataset(df):
    # Step 1: Remove duplicates
    df = df.drop_duplicates()

    # Step 2: Handle missing values
    df = df.fillna({
        'age': df['age'].median(),
        'income': df['income'].median(),
        'name': 'Unknown',
        'signup_date': pd.Timestamp('2024-01-01')
    })

    # Step 3: Fix data types
    df['age'] = df['age'].astype(int)
    df['signup_date'] = pd.to_datetime(df['signup_date'])

    # Step 4: Remove outliers
    q1 = df['income'].quantile(0.25)
    q3 = df['income'].quantile(0.75)
    iqr = q3 - q1
    df = df[(df['income'] >= q1 - 1.5*iqr) & (df['income'] <= q3 + 1.5*iqr)]

    return df
