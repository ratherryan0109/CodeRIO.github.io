C Programming

C is a powerful, low-level programming language that forms the foundation of modern computing. It provides direct memory access, making it ideal for system programming, embedded systems, and performance-critical applications.

Intermediate
10 Weeks
18 Modules

Overview

Created by Dennis Ritchie at Bell Labs in 1972, C is one of the most influential programming languages ever developed. It was designed for system programming and remains the gold standard for operating systems, embedded systems, and performance-critical software.

C provides low-level memory access through pointers, manual memory management, and minimal runtime overhead. Its influence extends to C++, C#, Java, Python, and many modern languages that borrow its syntax and concepts. The UNIX operating system was written in C, and Linux, Windows, and macOS kernels still use C extensively.

Despite being over 50 years old, C remains essential for systems programming, embedded devices (IoT), real-time systems, game engines, compilers, databases, and high-frequency trading. Understanding C provides deep insight into how computers work at the hardware level.

Why Learn C?

System-Level Access

Direct memory manipulation via pointers.

Maximum Performance

Minimal overhead, closest to machine code.

Foundation of CS

Understand how memory, stacks, and pointers work.

Embedded & IoT

Notes

1. Pointers

Pointers store memory addresses. They are essential for dynamic memory, arrays, and efficient parameter passing.

int x = 42;
int *ptr = &x;      // ptr holds address of x
printf("%d\n", *ptr); // 42 (dereference)
*ptr = 100;         // change x through ptr
printf("%d\n", x);  // 100

int arr[5] = {1,2,3,4,5};
int *p = arr;       // array decays to pointer
printf("%d", *(p+2)); // 3 (pointer arithmetic)

2. Dynamic Memory Allocation

int *arr = (int*)malloc(5 * sizeof(int));
if (arr == NULL) { /* handle error */ }
for (int i = 0; i < 5; i++) arr[i] = i * 10;

arr = (int*)realloc(arr, 10 * sizeof(int));
// now arr has space for 10 ints

free(arr); // always free!
arr = NULL; // avoid dangling pointer

3. Structures

typedef struct {
    char name[50];
    int age;
    float gpa;
} Student;

Student s1 = {"Alice", 20, 3.8};
Student *ptr = &s1;
printf("%s", ptr->name); // arrow for pointer
printf("%s", s1.name);   // dot for struct

4. File I/O

FILE *fp = fopen("data.txt", "w");
if (fp == NULL) { perror("Error"); return; }
fprintf(fp, "Name: %s, Age: %d\n", "Alice", 20);
fclose(fp);

fp = fopen("data.txt", "r");
char buffer[256];
while (fgets(buffer, sizeof(buffer), fp)) {
    printf("%s", buffer);
}
fclose(fp);

5. Function Pointers

int add(int a, int b) { return a + b; }
int sub(int a, int b) { return a - b; }

int (*operation)(int, int);
operation = add;
printf("%d\n", operation(5, 3)); // 8

operation = sub;
printf("%d\n", operation(5, 3)); // 2

// Array of function pointers
int (*ops[])(int,int) = {add, sub};

6. Bit Manipulation

unsigned char flags = 0;
#define FLAG_A 0b00000001
#define FLAG_B 0b00000010
#define FLAG_C 0b00000100

flags |= FLAG_A;         // set bit
flags &= ~FLAG_B;        // clear bit
flags ^= FLAG_C;         // toggle bit
int isSet = flags & FLAG_A;  // check bit

// Check if power of 2
int isPowerOf2(int n) {
    return n > 0 && (n & (n-1)) == 0;
}

Examples

Example: Linked List

typedef struct Node {
    int data;
    struct Node* next;
} Node;

Node* createNode(int data) {
    Node* n = (Node*)malloc(sizeof(Node));
    n->data = data; n->next = NULL;
    return n;
}

void append(Node** head, int data) {
    Node* n = createNode(data);
    if (*head == NULL) { *head = n; return; }
    Node* curr = *head;
    while (curr->next) curr = curr->next;
    curr->next = n;
}

void printList(Node* head) {
    while (head) {
        printf("%d -> ", head->data);
        head = head->next;
    }
    printf("NULL\n");
}
