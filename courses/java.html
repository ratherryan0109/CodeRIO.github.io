Overview

Java was developed by James Gosling at Sun Microsystems and released in 1995. It was designed with the principle of "write once, run anywhere" (WORA), meaning compiled Java code can run on any platform with a Java Virtual Machine (JVM) without recompilation.

Java is a statically-typed, class-based, object-oriented language that emphasizes early error detection and robust memory management through automatic garbage collection. Its ecosystem includes Spring Boot for enterprise apps, Android SDK for mobile, Hibernate for ORM, and Maven/Gradle for build automation.

With millions of developers worldwide, Java is used in banking systems, e-commerce platforms, scientific applications, big data processing (Hadoop, Spark), and cloud-native microservices. The language continues to evolve with bi-annual releases, adding features like lambda expressions (Java 8), modules (Java 9), records and pattern matching (Java 16+).

Notes

1. Java Main Method

The entry point of every Java application.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

2. Collections & Generics

Type-safe collections with generics.

List<String> names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
names.forEach(System.out::println);

Map<String, Integer> ages = new HashMap<>();
ages.put("Alice", 30);
ages.put("Bob", 25);

// Stream API
int sum = ages.values().stream()
    .mapToInt(Integer::intValue)
    .sum();

3. Lambda Expressions

// Functional interface
@FunctionalInterface
interface Calculator {
    int calculate(int a, int b);
}

Calculator add = (a, b) -> a + b;
Calculator multiply = (a, b) -> a * b;

System.out.println(add.calculate(5, 3));    // 8
System.out.println(multiply.calculate(5, 3)); // 15

4. Spring Boot REST Controller

@RestController
@RequestMapping("/api/users")
public class UserController {

    @GetMapping
    public List<User> getAllUsers() {
        return userService.findAll();
    }

    @PostMapping
    public User createUser(@RequestBody User user) {
        return userService.save(user);
    }

    @GetMapping("/{id}")
    public User getUserById(@PathVariable Long id) {
        return userService.findById(id);
    }
}

5. Threading Basics

// Using Thread class
class MyThread extends Thread {
    public void run() {
        System.out.println("Running in: " + Thread.currentThread().getName());
    }
}
new MyThread().start();

// Using Runnable (preferred)
Runnable task = () -> System.out.println("Task running");
new Thread(task).start();

// ExecutorService
ExecutorService executor = Executors.newFixedThreadPool(4);
executor.submit(() -> System.out.println("Pooled task"));
executor.shutdown();

6. Records (Java 16+)

// Compact data carrier class
public record Person(String name, int age) {}

// Usage
Person p = new Person("Alice", 30);
System.out.println(p.name());  // Alice
System.out.println(p.age());   // 30
System.out.println(p);         // Person[name=Alice, age=30]

// Records automatically generate: constructor, getters, equals, hashCode, toString

7. Exception Handling

public void readFile(String path) {
    try (BufferedReader br = new BufferedReader(new FileReader(path))) {
        String line;
        while ((line = br.readLine()) != null) {
            System.out.println(line);
        }
    } catch (FileNotFoundException e) {
        System.err.println("File not found: " + path);
    } catch (IOException e) {
        System.err.println("IO Error: " + e.getMessage());
    }
    // try-with-resources auto-closes br
}

8. JVM Memory Model

Heap stores objects, Stack stores primitives and references. Metaspace (replaced PermGen) stores class metadata. Garbage collection (GC) reclaims unused objects in the heap.

// JVM flags example
// -Xms512m -Xmx2g -XX:+UseG1GC
// -XX:MaxMetaspaceSize=256m

// Memory monitoring
Runtime rt = Runtime.getRuntime();
System.out.println("Total: " + rt.totalMemory());
System.out.println("Free: " + rt.freeMemory());
System.out.println("Max: " + rt.maxMemory());

Example: Spring Boot CRUD App

@Entity
public class Product {
    @Id @GeneratedValue
    private Long id;
    private String name;
    private double price;
    // getters, setters, constructors
}

@Repository
public interface ProductRepository extends JpaRepository<Product, Long> {
    List<Product> findByNameContaining(String keyword);
}

@Service
public class ProductService {
    @Autowired
    private ProductRepository repo;

    public List<Product> searchProducts(String q) {
        return repo.findByNameContaining(q);
    }
}
