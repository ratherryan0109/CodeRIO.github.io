Overview

Data Structures & Algorithms form the foundation of computer science. Data structures organize data efficiently for access and modification, while algorithms are step-by-step procedures for solving computational problems. Together they determine the efficiency, scalability, and correctness of software systems.

DSAs are evaluated through time complexity (how fast an algorithm runs) and space complexity (how much memory it uses), both expressed using Big O notation. Understanding these concepts enables you to choose the right tool for any problem - from storing millions of user records to finding the shortest path in a navigation system.

DSA knowledge is critical for interviews at top tech companies (Google, Microsoft, Amazon, Meta, Apple). It is universally applicable across programming languages and domains - whether you build web apps, mobile apps, games, or distributed systems. Mastery of DSA separates novice programmers from senior engineers.

Notes

Big O Cheatsheet

Array        O(1) access  O(n) search  O(n) insert/delete
Hash Map    O(1) average access/insert/delete
BST         O(log n) average for all operations
Heap        O(log n) insert  O(log n) delete  O(1) min/max
Graph       O(V+E) BFS/DFS  O(E log V) Dijkstra
Sorting     O(n log n) best comparison sorts

Recursion Template

function fn(n):
    if baseCondition(n): return baseValue
    return combine(fn(n-1), n)

DP Framework

  1. Define state (what does dp[i] represent?)
  2. Find recurrence relation (how does dp[i] relate to previous?)
  3. Set base cases
  4. Iterate/recursively compute
  5. Return final answer

Code Examples

Binary Search (O(log n))

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target: return mid
        elif arr[mid] < target: left = mid + 1
        else: right = mid - 1
    return -1

def inorder(root):

    inorder(root.left)
    print(root.val)
    inorder(root.right)

Dijkstra Shortest Path

import heapq
def dijkstra(graph, start):
    dist = {node: float("inf") for node in graph}
    dist[start] = 0
    pq = [(0, start)]
    while pq:
        d, u = heapq.heappop(pq)
        if d > dist[u]: continue
        for v, w in graph[u]:
            if d + w < dist[v]:
                dist[v] = d + w
                heapq.heappush(pq, (dist[v], v))
    return dist

Fibonacci with DP (tabulation)

def fib(n):
    if n <= 1: return n
    dp = [0] * (n + 1)
    dp[1] = 1
    for i in range(2, n + 1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]
